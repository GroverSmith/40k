Warhammer 40k Points Data Converter

How to use this converter:

  1. Open the PDF file: resources/points-v3_2-2025-08.pdf
  2. Copy the text content from the PDF (Ctrl+A, Ctrl+C)
  3. Paste the text into the textarea below
  4. Click "Convert to Structured Data"
  5. Download the result as CSV or JavaScript file
  6. Use the JavaScript file to update your points data

Note: The converter will automatically detect factions, units, and enhancements based on common patterns in the PDF.
